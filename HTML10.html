Project Ten Div and Spans Block and Inline Elements:-

A container in html is basically a set of tags that grabs other html elements. As You go into more advance html you need to use these containers for CSS in your website.

Basically There are two famous cointainers for this purpose Div and Spans.

html has two basic ways to display elements. These are block and inline.

Block elements are elements that take up the entire width of the page like a block on a page.

Inline elements are Elements that take only that much of space that they need like a link in a page.

You can have Inline elements sitting right to each other. But you can not have a block element sit right next to eachother.Because they take the intire width of page.

Link which is Inline Element in a page is able to sit right next to each other.Because they need as much space as the need but not a block. Because it needs the whole width of the web page.

Difference between a Div and a span

Spans are InLine Containers and Divs are block containers.
